## Suponga que tiene las tablas *Ciudad* y *Pais* compuestas de la siguiente forma.  

### Ciudad                 

| `PK` | ID | int |
|--------|--------|:--------:|
|  | Nombre | nvarchar(100) |
| `FK1`  | Id_Pais  | int |
|  | Población | int |
  

### Pais
  
| `PK` | ID | int |
|--------|:--------|:--------:|
|  | Nombre | nvarchar(100) |
|  | Continente  | nvarchar(100) |
|  | PBI | int |
  
Note que existe una relacion entre *`FK1` - Id_pais* de la tabla Ciudad y *`PK`- ID* de la tabla Pais
